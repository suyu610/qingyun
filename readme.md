## 关于青云知识库
一个微信小程序，用于学生之间资料共享。
希望能以微薄力量解决中文社区信息密度不足，资料良莠不齐的问题。
### 主要功能
资料上传、购买（微信支付）、小程序内浏览文档、网页版浏览文档、提问、管理

### 技术栈
Springboot、微信小程序（[Vant](https://vant-contrib.gitee.io/vant-weapp/#/home "Vant")、[ColorUI](https://www.color-ui.com/ "ColorUI"))、H5、Redis、MySQL、
采用腾讯云COS作为网盘，控制读写权限
