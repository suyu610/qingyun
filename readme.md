## 关于青云知识库
一个微信小程序，用于青岛大学学生之间进行资料分享。
- 主要功能为：资料上传、购买（微信支付）、预览文档、网页版预览文档、提问、管理
- 使用的技术栈为：Springboot、微信小程序（[Vant](https://vant-contrib.gitee.io/vant-weapp/#/home "Vant")、[ColorUI](https://www.color-ui.com/ "ColorUI"))、H5、Redis、MySQL
- 采用腾讯云COS作为网盘，控制读写权限
